2.17 Interruptor de Inclinación
=================================

Resumen
----------

En esta lección, aprenderás sobre el interruptor de inclinación. Este tipo 
de interruptor puede usarse para detectar si un objeto está inclinado, lo 
cual tiene gran valor en aplicaciones prácticas. Se utiliza, por ejemplo, 
para medir la inclinación de puentes, edificios, torres de transmisión, 
entre otros, y juega un rol importante en tareas de mantenimiento preventivo.

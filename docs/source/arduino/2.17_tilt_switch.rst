2.17 Interrupteur d'inclinaison
==================================

Aperçu
----------------

Dans cette leçon, vous allez découvrir l'interrupteur d'inclinaison. 
Ce composant peut être utilisé pour détecter l'inclinaison d'objets, 
ce qui est d'une grande utilité dans les applications pratiques. Il peut, 
par exemple, permettre de surveiller l'inclinaison de ponts, de bâtiments, 
de pylônes de lignes de transmission, et ainsi jouer un rôle important pour 
guider les travaux de maintenance.

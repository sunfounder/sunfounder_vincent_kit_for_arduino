2.18 Module d'interrupteur tactile
=====================================

Aperçu
------------

Dans cette leçon, vous allez découvrir le module d'interrupteur tactile. 
Ce module peut remplacer les interrupteurs traditionnels, offrant les 
avantages suivants : une utilisation pratique, une sensation tactile agréable, 
un contrôle précis et une usure mécanique minimale.

Composants Requis
------------------------

.. image:: img/Part_two_18.png


* :ref:`cpn_mega2560`
* :ref:`cpn_wires`
* :ref:`cpn_touch`

d'interrupteur tactile.

Après avoir téléversé le code sur la carte Mega2560, vous pourrez voir les 
lectures des broches affichées sur le moniteur série.

Lorsque vous touchez le module tactile avec le bout de votre doigt, un 「1」 
s'affiche sur le moniteur série ; lorsque vous retirez votre doigt, un 「0」 
s'affiche. Pour une explication détaillée du code, référez-vous à :ref:`ar_digital_read`.

2.18 Modulo Interruttore Touch
=================================

Panoramica
-------------

In questa lezione conoscerai il modulo interruttore touch. Questo modulo 
può sostituire i tradizionali tipi di interruttori grazie a diversi vantaggi: 
operazione semplice, sensazione al tocco piacevole, controllo preciso e minima 
usura meccanica.

Componenti necessari
------------------------

.. image:: img/Part_two_18.png

* :ref:`cpn_mega2560`
* :ref:`cpn_wires`
* :ref:`cpn_touch`

Circuito Fritzing
---------------------

interruttore touch.

Dopo aver caricato il codice sulla scheda Mega2560, potrai vedere le letture 
dei pin visualizzate sul monitor seriale.

Quando la punta del dito tocca il modulo interruttore touch, sul monitor 
seriale verrà visualizzato 「1」; quando rimuovi il dito, verrà visualizzato 「0」. 
Per una spiegazione dettagliata del codice, consulta :ref:`ar_digital_read`.

2.18 Módulo de Interruptor Táctil
=====================================

Resumen
----------

En esta lección, aprenderás sobre el módulo de interruptor táctil. Este 
puede reemplazar los interruptores tradicionales con ventajas como operación 
conveniente, agradable al tacto, control preciso y mínimo desgaste mecánico.

Componentes necesarios
-------------------------

.. image:: img/Part_two_18.png

* :ref:`cpn_mega2560`
* :ref:`cpn_wires`
* :ref:`cpn_touch`

los pines en el monitor serial.

Cuando tu dedo toca el módulo de interruptor táctil, el monitor serial mostrará 「1」; 
cuando retiras el dedo, se mostrará 「0」. Para una explicación detallada del 
código, consulta :ref:`ar_digital_read`.

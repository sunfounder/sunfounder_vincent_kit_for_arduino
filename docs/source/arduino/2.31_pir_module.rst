2.31 Module PIR
=================

Aperçu
----------

Dans cette leçon, vous apprendrez à utiliser le module PIR. Le capteur PIR 
détecte les radiations thermiques infrarouges ou la présence d'organismes 
émettant des radiations thermiques infrarouges. Ce module est largement 
utilisé dans la vie quotidienne, notamment pour les alarmes anti-intrusion 
et les dispositifs de notification de présence.

Composants requis
-------------------

.. image:: img/Part_two_31.png

* :ref:`cpn_mega2560`
* :ref:`cpn_wires`
* :ref:`cpn_pir`

Circuit Fritzing
-----------------

Dans cet exemple, nous pouvons connecter directement les broches du module 
PIR aux broches de la carte Mega 2560. Nous utilisons la broche numérique 2 
pour lire le signal du module PIR. Connectez le VCC du module PIR à 5V, GND 
à GND et OUT à la broche numérique. NOTE : vous pouvez retirer le capot du 
module PIR pour voir l'étiquette de la broche.

Une fois le code téléchargé sur la carte Mega2560, vous pouvez ouvrir le 
moniteur série pour voir la valeur lue sur la broche. Lorsque le module 
PIR détecte une activité à proximité, le moniteur série affichera « 1 » ; 
sinon, il affichera « 0 ». Consultez :ref:`ar_digital_read` pour plus de 
détails sur l'explication du code.

Le module PIR dispose de deux potentiomètres : l'un pour ajuster la 
**sensibilité** et l'autre pour régler la **distance de détection**. 
Pour améliorer le fonctionnement du module PIR, il est recommandé 
d'ajuster ces deux potentiomètres.

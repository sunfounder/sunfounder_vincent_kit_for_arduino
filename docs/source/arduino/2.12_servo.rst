2.12 Servo
============

Panoramica
-------------

In questa lezione imparerai a conoscere il Servo. Il Servo è un tipo di 
attuatore che permette di regolare e mantenere la posizione angolare o un 
attuatore rotativo che consente un controllo preciso della posizione angolare. 
Attualmente, è ampiamente utilizzato in giocattoli telecomandati di fascia 
alta, come aeroplani, sottomarini, telerobot e così via.

Analisi del codice
----------------------

Utilizzando la libreria Servo.h, puoi controllare facilmente il servo.

.. code-block:: arduino

    #include <Servo.h> 

**Funzioni della libreria:**

.. code-block:: arduino

    Servo

Crea un oggetto **Servo** per controllare un servo.

.. code-block:: arduino

    uint8_t attach(int pin); 

Assegna un pin come driver del servo. Chiama pinMode. Restituisce 0 in caso di errore.

.. code-block:: arduino

    void detach();

Libera un pin dal controllo del servo.

.. code-block:: arduino

    void write(int value); 

Imposta l'angolo del servo in gradi, da 0 a 180.

.. code-block:: arduino

    int read();

Restituisce il valore impostato con l'ultima chiamata a write().

.. code-block:: arduino

    bool attached(); 

Restituisce 1 se il servo è attualmente collegato.

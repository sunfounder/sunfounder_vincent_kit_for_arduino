2.12 Servo
============

Aperçu
-----------

Dans cette leçon, vous allez découvrir le servo-moteur. Un servo est un 
dispositif permettant de contrôler de manière précise une position angulaire 
qui peut être ajustée et maintenue. Il est couramment utilisé dans les 
jouets télécommandés haut de gamme, tels que les avions, sous-marins, robots, etc.

Analyse du Code
------------------

En utilisant la bibliothèque Servo.h, vous pouvez piloter facilement le servo.

.. code-block:: arduino

    #include <Servo.h> 

**Fonctions de la Bibliothèque：**

.. code-block:: arduino

    Servo

Créer un objet **Servo** pour contrôler un servo.

.. code-block:: arduino

    uint8_t attach(int pin); 

Configurer une broche en tant que pilote de servo. Appelle pinMode. Renvoie 0 en cas d'échec.

.. code-block:: arduino

    void detach();

Libère une broche de la commande du servo.

.. code-block:: arduino

    void write(int value); 

Définit l'angle du servo en degrés, de 0 à 180.

.. code-block:: arduino

    int read();

Renvoie la dernière valeur définie avec write().

.. code-block:: arduino

    bool attached(); 

Renvoie 1 si le servo est actuellement attaché.

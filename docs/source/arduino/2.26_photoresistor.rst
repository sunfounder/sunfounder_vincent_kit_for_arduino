2.26 Photoresistor
====================

Aperçu
--------

Dans cette leçon, vous apprendrez à utiliser la photo-résistance. 
La photo-résistance est utilisée dans de nombreux appareils électroniques, 
tels que les exposimètres de caméra, les radios-réveils, les dispositifs 
d'alarme (comme détecteur de faisceau), les petites veilleuses, les horloges 
extérieures, les lampes de rue solaires, etc. Elle est placée dans les 
réverbères pour contrôler l'allumage de la lumière. La lumière ambiante tombant 
sur la photo-résistance provoque l'allumage ou l'extinction des lampes de rue.

Circuit Fritzing
--------------------

Dans cet exemple, nous utilisons la broche analogique (A0) pour lire la 
valeur de la photo-résistance. Une broche de la photo-résistance est 
connectée à 5V, l'autre est reliée à A0. De plus, une résistance de 10kΩ 
est nécessaire avant de connecter l'autre broche à GND.

Après avoir téléchargé le code sur la carte Mega2560, vous pouvez ouvrir le 
moniteur série pour voir la valeur lue de la broche. Lorsque la lumière 
ambiante devient plus forte, la lecture augmentera en conséquence. La plage de 
lecture de la broche est de « 0 » à « 1023 ». Cependant, selon les conditions 
environnementales et les caractéristiques de la photo-résistance, la plage de 
lecture réelle peut être plus petite que la plage théorique. Pour une explication 
détaillée du code, consultez :ref:`ar_analog_read`.

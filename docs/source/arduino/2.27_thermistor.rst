2.27 Termistor
================

Resumen
----------

En esta lección, aprenderás cómo utilizar un termistor. Este componente 
se puede emplear como parte de circuitos electrónicos para compensación 
de temperatura en instrumentos de medición. Se utiliza en dispositivos 
como amperímetros, caudalímetros, analizadores de gas, y otros. También 
puede usarse para protección contra sobrecalentamiento, relés sin contacto, 
control de temperatura constante, control de ganancia automático, arranque 
de motores, retardo de tiempo, desmagnetización automática en televisores 
de color, alarmas contra incendios y compensación de temperatura.

Circuito Fritzing
------------------

En este ejemplo, utilizamos el pin analógico A0 para obtener el valor del 
termistor. Un pin del termistor se conecta a 5V y el otro a A0. Además, una 
resistencia de 10kΩ se conecta entre el otro pin del termistor y GND.

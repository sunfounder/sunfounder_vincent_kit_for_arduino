2.22 Potenciómetro
====================

Resumen
---------

En esta lección, aprenderás sobre el potenciómetro. El potenciómetro es un 
componente resistivo de 3 terminales cuya resistencia se puede ajustar según 
una variación regular.

Componentes necesarios
--------------------------

.. image:: img/Part_two_22.png

* :ref:`cpn_mega2560`
* :ref:`cpn_breadboard`
* :ref:`cpn_wires`
* :ref:`cpn_potentiometer`

Circuito Fritzing
----------------------

.. image:: img/Part_two_22_Circuit.png
   :align: center

En este ejemplo, usamos el pin analógico (A0) para leer el valor del 
potenciómetro. Al girar el eje del potenciómetro, puedes cambiar la 
distribución de resistencia entre estos tres pines, alterando así el 
voltaje en el pin central. Cuando la resistencia entre el pin central 
y un pin externo conectado a 5V es cercana a cero (y la resistencia 
entre el pin central y el otro pin externo es cercana a 10kΩ), el voltaje 
en el pin central es cercano a 5V. La operación inversa (resistencia 
cercana a 10kΩ entre el pin central y el pin externo conectado a 5V) hará 
que el voltaje en el pin central se acerque a 0V.

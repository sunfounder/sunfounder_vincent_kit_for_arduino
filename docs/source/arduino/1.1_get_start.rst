1.1 Comenzando con el IDE de Arduino
==========================================

Arduino es una plataforma de código abierto con software y hardware sencillos. 
Puedes aprender a utilizarla en poco tiempo, incluso si eres principiante. 
Ofrece un entorno de desarrollo integrado (IDE) para compilar el código, 
compatible con múltiples placas de control. Así que solo necesitas descargar 
el IDE de Arduino, cargar los bocetos (es decir, los archivos de código) en la 
placa y luego podrás observar los fenómenos experimentales correspondientes. 
Para más información, consulta http://www.arduino.cc.

Sigue el tutorial a continuación para aprender cómo instalar el IDE de Arduino, agregar bibliotecas y cargar el código.

1.1 Premiers pas avec l'Arduino IDE
======================================

Arduino est une plateforme open source avec des logiciels et du matériel 
simples à utiliser. Vous pouvez la maîtriser rapidement, même si vous êtes 
débutant. Elle propose un environnement de développement intégré (IDE) pour 
la compilation de code, compatible avec plusieurs cartes de contrôle. Il vous 
suffit donc de télécharger l'IDE Arduino, de téléverser les sketches (c'est-à-dire 
les fichiers de code) sur la carte, et vous pourrez observer les phénomènes 
expérimentaux correspondants. Pour plus d'informations, consultez http://www.arduino.cc.

Veuillez suivre le tutoriel ci-dessous pour apprendre à installer l'IDE Arduino, ajouter des bibliothèques et téléverser du code.

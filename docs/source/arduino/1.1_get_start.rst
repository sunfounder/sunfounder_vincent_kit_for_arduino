1.1 Introduzione ad Arduino IDE
===============================

Arduino è una piattaforma open source con hardware e software semplici. 
Anche se sei un principiante, puoi imparare a usarla in poco tempo. Fornisce 
un ambiente di sviluppo integrato (IDE) per la compilazione del codice, compatibile 
con diverse schede di controllo. Ti basta scaricare Arduino IDE, caricare gli sketch 
(ovvero i file di codice) sulla scheda e osservare i fenomeni sperimentali correlati. 
Per maggiori informazioni, visita http://www.arduino.cc.

Segui il tutorial qui sotto per imparare a installare Arduino IDE, aggiungere librerie e caricare il codice.

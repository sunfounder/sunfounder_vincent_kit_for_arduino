2.10 Buzzer Actif
====================

Aperçu
----------

Dans cette leçon, vous allez découvrir le buzzer actif. Ce type de buzzer 
électronique avec une structure intégrée est alimenté par une source de 
courant continu (DC). Le buzzer actif est largement utilisé dans les ordinateurs, 
alarmes, jouets électroniques, téléphones, minuteries, et autres produits 
électroniques ou dispositifs sonores.

Composants Requis
-------------------

.. image:: img/list_2.10.png

* :ref:`cpn_mega2560`
* :ref:`cpn_breadboard`
* :ref:`cpn_wires`
* :ref:`cpn_buzzer`


Circuit Fritzing
------------------

.. image:: img/image448.png

Dans cet exemple, nous utilisons la broche numérique 9 pour piloter le buzzer. 
Connectez la cathode du buzzer à la masse (GND) et l'anode à la broche numérique 9.

Gráfico de Barras LED
=========================

.. image:: img/bar_graph.png
    :width: 300
    :align: center

El gráfico de barras LED es una matriz de LEDs que se conecta a un circuito 
electrónico o microcontrolador. Es fácil conectar un gráfico de barras LED 
al circuito, de manera similar a conectar 10 LEDs individuales a 10 pines de 
salida. Generalmente, se puede usar el gráfico de barras LED como indicador 
de nivel de batería, en equipos de audio y paneles de control industriales. 
Hay muchas otras aplicaciones para los gráficos de barras LED.

A continuación, se muestra el **Esquema Interno** del gráfico de barras LED. 
Generalmente, el lado con la etiqueta es el ánodo y el otro lado es el cátodo.

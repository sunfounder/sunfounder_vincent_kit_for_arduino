Fils de Liaison (Jumpers)
===============================

Les fils qui relient deux terminaux sont appelés fils de liaison ou jumpers. 
Il existe divers types de jumpers. Ici, nous nous concentrons sur ceux utilisés 
sur une plaque d'essai. Parmi leurs fonctions, ils servent à transmettre les 
signaux électriques d'un point de la plaque d'essai vers les broches d'entrée/sortie 
d'un microcontrôleur.

Les jumpers se fixent en insérant leurs "connecteurs d'extrémité" dans les 
emplacements de la plaque d'essai, sous lesquels se trouvent des plaques 
parallèles connectant les emplacements en groupes de rangées ou de colonnes 
selon la zone. Les "connecteurs d'extrémité" sont insérés dans la plaque sans 
besoin de soudure, dans les emplacements spécifiques à connecter pour le prototype.

Il existe trois types de jumpers : Femelle-à-Femelle, Mâle-à-Mâle et Mâle-à-Femelle. 
Le terme Mâle-à-Femelle signifie qu’un côté est doté d’une fiche mâle et l’autre d’une 
prise femelle. Mâle-à-Mâle désigne deux côtés mâles et Femelle-à-Femelle signifie que 
les deux extrémités sont femelles.

.. image:: img/image414.png


Il est possible d’utiliser plusieurs types de jumpers dans un même projet. 
**Les couleurs des jumpers varient pour faciliter l'identification des 
connexions entre les circuits ; elles n’ont pas de fonction spécifique**.

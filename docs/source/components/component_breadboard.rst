Breadboard
==============

.. image:: img/breadboard.png
    :width: 600

Una breadboard è una base di costruzione utilizzata per prototipare circuiti elettronici. In origine, il termine si riferiva a una tavola di legno levigata usata per tagliare il pane. Negli anni '70, divenne disponibile la breadboard senza saldature (nota anche come plugboard o terminal array board), e oggi il termine "breadboard" è comunemente utilizzato per riferirsi a questa tipologia.

È utilizzata per costruire e testare rapidamente i circuiti prima di completarne il design definitivo. 
Dispone di numerosi fori nei quali possono essere inseriti componenti come IC, resistori e fili jumper. 
La breadboard consente di collegare e rimuovere facilmente i componenti. 

L'immagine mostra la struttura interna di una breadboard. 
Sebbene questi fori sulla breadboard sembrino indipendenti, in realtà sono collegati internamente tramite strisce metalliche.

.. image:: img/breadboard_internal.png
    :width: 600

Protoboard
=============

.. image:: img/breadboard.png
    :width: 600

Una protoboard es una base de construcción utilizada para la creación y prueba de prototipos electrónicos. Originalmente, la palabra hacía referencia a una tabla literal para cortar pan, una pieza de madera pulida utilizada para rebanar pan. En la década de 1970, la protoboard sin soldadura (también conocida como plugboard o terminal array board) se hizo disponible y, hoy en día, el término "protoboard" se usa comúnmente para referirse a estas.

Se utiliza para construir y probar circuitos rápidamente antes de finalizar cualquier diseño de circuito. 
Tiene numerosos agujeros en los que se pueden insertar componentes como circuitos integrados (ICs), resistencias y cables de puente (jumper wires). 
La protoboard permite enchufar y retirar componentes con facilidad. 

La imagen muestra la estructura interna de una protoboard. 
Aunque los agujeros en la protoboard parecen ser independientes entre sí, en realidad están conectados internamente mediante tiras de metal.

.. image:: img/breadboard_internal.png
    :width: 600

Si deseas saber más sobre la protoboard, consulta: `Cómo usar una protoboard - Science Buddies <https://www.sciencebuddies.org/science-fair-projects/references/how-to-use-a-breadboard#pth-smd>`_

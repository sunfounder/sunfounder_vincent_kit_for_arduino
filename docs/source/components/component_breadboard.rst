Plaque de montage
=====================

.. image:: img/breadboard.png
    :width: 600

Une plaque de montage est une base de construction utilisée pour le prototypage électronique. À l'origine, le terme désignait une planche de bois polie utilisée pour couper le pain. Dans les années 1970, la plaque de montage sans soudure (également appelée plaque à broches ou planche de bornes) est devenue disponible, et aujourd'hui le terme « plaque de montage » est couramment utilisé pour désigner ces dispositifs.

Elle est utilisée pour construire et tester rapidement des circuits avant de finaliser 
tout design de circuit. Elle possède de nombreux trous dans lesquels les composants 
mentionnés ci-dessus, comme les circuits intégrés (CI) et les résistances, ainsi que les fils de connexion, peuvent être insérés. La plaque de montage permet d'insérer et de retirer facilement les composants.

L'image montre la structure interne d'une plaque de montage. Bien que ces trous sur la plaque semblent être indépendants les uns des autres, ils sont en réalité connectés entre eux par des bandes métalliques internes.

.. image:: img/breadboard_internal.png
    :width: 600

Protoboard
===========================

.. image:: img/image421.png

El protoboard facilita el diseño de circuitos personalizados. Puedes soldar 
componentes en el área de prototipos para crear tu proyecto o usarlo con una 
pequeña protoboard para probar rápidamente ideas de circuitos sin necesidad de 
soldadura. Tiene conexiones adicionales para todos los pines de E/S de Arduino 
y espacio para montar circuitos integrados de orificio pasante y de montaje en 
superficie. Es una forma conveniente de convertir tu circuito personalizado y 
Arduino en un solo módulo.

**Especificaciones**

-  Puede superponerse directamente en la placa Sunfounder Mega2560. Se proporciona 
   una pequeña protoboard que puedes usar para realizar algunos experimentos sencillos.

-  Incluye una huella para SOP28.

-  Tiene conexiones adicionales para todos los pines de E/S de Arduino.

-  Ofrecemos una almohadilla de conexión para los pines 22-53 que puedes utilizar 
   para soldar componentes directamente.
